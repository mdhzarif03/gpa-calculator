## ? Features
* **Intelligent Logic:** Automatically handles Higher Math vs. Biology vs. Agriculture swaps for Science students.
* **Group Support:** Full subject lists for Science, Commerce, and Humanities.
* **Modern UI:** Built with a clean, 'Sunset Violet' glassmorphism design.
* **Responsive:** Works perfectly on mobile, tablets, and desktops.
